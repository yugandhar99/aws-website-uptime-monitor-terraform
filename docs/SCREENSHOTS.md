# Screenshots to Add After Deployment

Add screenshots to `static/ss/` after you deploy the project.

Recommended screenshots:

1. GitHub repository home page
2. Terraform plan output
3. Terraform apply output
4. Lambda uptime checker configuration
5. EventBridge schedule
6. DynamoDB uptime records
7. CloudWatch custom metrics
8. CloudWatch alarms
9. SNS email alert
10. API Gateway `/health` response
11. React dashboard through CloudFront
12. GitHub Actions workflow success
13. Trivy/Checkov scan output
14. Optional GenAI incident summary output

Do not include screenshots showing AWS account IDs, access keys, private URLs, personal emails, or secrets.
